# AGENTS.md 作成計画

## 目的

このリポジトリのコントリビューター向けガイドとして、`AGENTS.md` を作成する。

## 修正方針

1. タイトルは `Repository Guidelines` とする。
2. Go CLI プロジェクトの現在の構成に合わせて、プロジェクト構造を説明する。
3. `go run .`、`go build`、`go test ./...` などの主要コマンドを記載する。
4. Go の標準的なスタイル、`gofmt`、package 分離方針、テスト命名規則を記載する。
5. 直近の Git 履歴に合わせ、日本語で簡潔なコミットタイトルを推奨する。
6. PR に必要な確認事項を簡潔に記載する。
7. 全体は 200-400 words 程度の英語 Markdown とし、説明は短く具体的にする。

## 想定編集ファイル

- `AGENTS.md`

## 確認方法

- `AGENTS.md` の内容確認
- `wc -w AGENTS.md` による語数確認
